# CareerPilot AI

CareerPilot AI is a production-ready Full Stack Agentic AI web application that helps users improve their resumes, receive AI-powered resume analysis, and discover suitable career opportunities based on their skills and experience.

## Live Links

### Client
https://career-client-seven.vercel.app

### Server
https://career-server-lake.vercel.app/api/v1

---

## Features

- AI Resume Analysis
- AI Career Recommendation Engine
- Resume Management
- Career Match Dashboard
- Authentication with JWT
- Role-based Dashboard
- Admin Panel
- User Profile Management
- Search & Filtering
- Responsive Design
- Dark Mode
- Toast Notifications
- Skeleton Loading
- About Page
- Contact Page
- Blog Page
- Support Page
